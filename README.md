# Qalby2Heart

Your Faith-Based Mental Wellness Companion

A Flutter mobile application designed to provide faith-based mental wellness support through:

- **AI Chat**: Connect with an AI Islamic counselor for faith-based guidance
- **Mood Tracking**: Monitor your emotional well-being with faith-based insights
- **Islamic Resources**: Access guidance from Quran and Hadith for mental wellness
- **Private Journal**: Express your thoughts in a safe, confidential space

## Features

- Clean, modern UI with green and white color scheme
- Bottom navigation for easy access to all features
- Privacy-focused design with secure storage
- Faith-based content and guidance
- Mood tracking with intensity levels
- Journaling with inspirational prompts

## Getting Started

1. Make sure you have Flutter installed on your system
2. Clone this repository
3. Run `flutter pub get` to install dependencies
4. Run `flutter run` to launch the app

## Project Structure

```
lib/
├── main.dart              # Main app entry point
└── screens/
    ├── home_screen.dart      # Home screen with quick access
    ├── ai_chat_screen.dart   # AI chat interface
    ├── mood_screen.dart      # Mood tracking screen
    ├── resources_screen.dart # Islamic resources
    └── journal_screen.dart   # Journal (write/history)
```
